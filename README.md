# NNBasicKnowledge

用 NumPy 从零手写神经网络基础模块，不依赖任何深度学习框架。

---

## LinearRegressionNum.py

从零实现线性回归，涵盖完整的训练流程：

1. **初始化参数** — 随机初始化权重 W，偏置 b 设为 0
2. **前向传播** — 用当前参数对输入 X 做预测，得到 y_hat
3. **计算损失** — 用均方误差（MSE）衡量预测值和真实值的差距
4. **反向传播** — 根据损失计算 W 和 b 各自的梯度
5. **参数更新** — 用梯度下降把参数往损失减小的方向移动
6. **循环迭代** — 重复以上步骤直到损失收敛

示例数据使用 8 个特征、200 个样本，训练结束后学到的参数会非常接近预设的真实值。

### 运行方式

```bash
python3 LinearRegressionNum.py
```

依赖：`numpy`
